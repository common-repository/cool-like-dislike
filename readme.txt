== Description ==

Плагин для WordPress, который поможет организовать систему кнопок "Нравится" и "Не нравится" на вашем сайте

== Installation ==

1. Загрузить содержимое папки `cool-like-dislike` в дирикторию `/wp-content/plugins/` вашего сайта.
2. Активировать плагин через меню 'Plugins' в WordPress.
3. Перейти к виджетам, выберать виджет под названием "Like - DisLike" и установить его в нужное место в вашей одиночной записи
